# DigiTools Platform

## Live Project Goals
- Build a clean Figma-inspired marketplace interface.
- Practice React state management with cart functionality.


## Technologies Used
- React.js
- Vite
- Tailwind CSS
- DaisyUI
- React Toastify


## Main Features
1. Product and cart toggle system with default product view.
2. Add to cart, remove from cart, and proceed to checkout functionality.
3. Responsive sections for banner, stats, steps, pricing, and footer.

## Getting Started
```bash
npm install
npm run dev
```

## Build for Production
```bash
npm run build
```
